# kpopdigest

Backend and frontend experiments for the K-pop Digest project.

## Frontend

- Location: `frontend/`
- Stack: Next.js (App Router) with Postgres access via `pg`
- Commands:
  - `npm install` (inside `frontend/`)
  - `npm run dev`
- Requires `DATABASE_URL` in `frontend/.env` (created automatically from `backend/.env` during setup)
